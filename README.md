# Многопоточный парсер вакансий

Проект для сбора информации о вакансиях с различных сайтов (hh.ru, SuperJob, Habr Career и др.) с использованием многопоточности.

## Возможности

- Парсинг вакансий с нескольких платформ одновременно
- Многопоточная обработка для ускорения работы
- Экспорт в JSON и CSV форматы
- Фильтрация и поиск вакансий
- Поддержка:
  - HeadHunter (hh.ru)
  - SuperJob
  - Habr Career
